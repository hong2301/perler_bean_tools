## 🚀 快速开始

### 方式一：一键打包（推荐）

双击运行项目根目录下的 `build_windows.bat`，等待打包完成即可。

```
📁 项目目录
├── build_windows.bat      ← 双击运行这个
├── launcher.py            ← 启动入口
├── app.py                 ← Flask 主程序
└── ...
```

### 方式二：手动打包

```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 执行打包命令
pyinstaller bead_recognizer.spec --clean --noconfirm

# 3. 输出文件在 dist/拼豆识别系统/ 目录
```

## 📦 详细打包步骤

### 1. 环境准备

确保 Windows 电脑已安装：

- **Python 3.8+** （推荐 3.10 或 3.11）
- **pip** （Python 包管理器）

```bash
# 验证安装
python --version
pip --version
```

### 2. 安装项目依赖

```bash
# 在项目根目录执行
pip install -r requirements.txt
```

**注意**：PaddleOCR 依赖较大（约 500MB+），首次安装可能需要较长时间。

### 3. 运行一键打包脚本

```bash
# 方式 1：双击 build_windows.bat
# 方式 2：命令行执行
build_windows.bat
```

打包过程约需 **5-15 分钟**，取决于电脑性能。

### 4. 获取输出文件

打包完成后，输出目录结构：

```
dist/拼豆识别系统/
├── 拼豆识别系统.exe      ← 主程序（双击运行）
├── _internal/            ← 依赖文件（不要删除）
│   ├── python3.dll
│   ├── templates/        ← HTML 模板
│   ├── uploads/          ← 上传目录
│   └── ...
└── ...
```

**使用方法**：
1. 将整个 `dist/拼豆识别系统` 文件夹复制到目标电脑
2. 双击运行 `拼豆识别系统.exe`
3. 程序会自动打开浏览器

## ❓ 常见问题

### Q1: 打包后程序体积过大（>1GB）

**原因**：PaddleOCR 和深度学习模型本身较大。

**解决方案**：
```python
# 在 bead_recognizer.spec 中添加排除项
excludes = [
    'matplotlib',
    'tkinter',
    # ... 其他不需要的库
]
```

### Q2: 运行时提示缺少 DLL

**解决方案**：
1. 安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. 确保打包时包含所有依赖：
```bash
pyinstaller bead_recognizer.spec --clean
```

### Q3: 浏览器无法自动打开

**解决方案**：
- 手动访问：`http://127.0.0.1:8080`
- 检查端口 8080 是否被占用（程序会自动尝试其他端口）

### Q4: OCR 识别失败

**原因**：可能是模型文件未正确打包。

**检查**：
```bash
# 确保 _internal 目录包含 paddleocr 相关文件
dist/拼豆识别系统/_internal/
├── paddleocr/
└── paddle/
```

### Q5: 打包时出现 "Analysis-00.toc not found"

**解决方案**：
```bash
# 清理缓存后重新打包
pyinstaller --clean --noconfirm bead_recognizer.spec
```

## ⚙️ 高级配置

### 单文件模式（.exe 只有一个文件）

编辑 `bead_recognizer.spec`：

```python
# 找到 EXE 配置，修改如下
exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.zipfiles,
    a.datas,
    [],
    name='拼豆识别系统',
    debug=False,
    bootloader_ignore_signals=False,
    strip=False,
    upx=True,
    upx_exclude=[],
    runtime_tmpdir=None,
    console=True,
    # 单文件模式关键参数
    onefile=True,  # 添加这一行
)
```

**注意**：单文件模式启动较慢，不推荐用于此项目。

### 添加程序图标

1. 准备 `.ico` 格式的图标文件（如 `icon.ico`）
2. 修改 `bead_recognizer.spec`：

```python
exe = EXE(
    ...
    icon='icon.ico',  # 取消注释并指定路径
    ...
)
```

### 隐藏控制台窗口

编辑 `bead_recognizer.spec`：

```python
exe = EXE(
    ...
    console=False,  # 改为 False
    ...
)
```

**警告**：隐藏控制台后将无法看到错误信息，仅建议在稳定版本使用。
